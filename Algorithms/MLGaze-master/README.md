# MLGaze
A machine learning repository for pattern recognition of gaze errors in consumer eye trackers.
The MLGaze repository contains python coding resources for gaze data processing such as data augmentation, outlier removal , creation of gaze features for training and machine learning algorithms for classification and modelling of gaze error patterns. The data used for implementing and testing the methods may be found in the Mendeley repository (link below).

https://data.mendeley.com/datasets/cfm4d9y7bh/1
